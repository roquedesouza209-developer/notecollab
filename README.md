# NoteCollab

A real-time collaborative notes platform built for fast, simple, and shared writing.


## 🚀 Overview

NoteCollab allows users to write, save, and collaborate on notes in real time.

It focuses on creating a lightweight system for shared productivity without unnecessary complexity.


## ✨ Core Features

- Real-time note syncing across clients  
- Autosave and manual save  
- Export notes as `.txt` or `.pdf`  
- Word count and active viewer tracking  
- Multiple themes and UI customization  
- Built-in support system  


## 🧠 System Design

- Real-time updates using Server-Sent Events  
- SQLite-backed support system  
- Lightweight architecture for speed and simplicity  


## 🛠 Tech Stack

- Node.js  
- SQLite  
- JavaScript, HTML, CSS  


## ⚙️ Run Locally

```bash
npm run dev
